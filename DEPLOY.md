# 性感贪吃蛇 H5 部署指南

## 1. 立即执行部署
在 SexySnakeH5 目录下运行：
`npm install && npm run build`

## 2. GitHub Pages 配置
只要你把 `dist` 目录的内容推送到 GitHub 的 `gh-pages` 分支，
你的游戏就会出现在：`https://[你的用户名].github.io/sexy-snake/`

## 3. 流量收割文案 (已准备好)
我已经在 `marketing/viral_h5_post.md` 为你写好了 3 种不同风格的推文。
- 风格 A: 悬念型 (为什么这个蛇会喘气？)
- 风格 B: 挑战型 (你能坚持到第 50 分钟解锁全部音效吗？)
- 风格 C: 宅男福利型 (深夜戴耳机专供)
